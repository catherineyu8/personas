Observations

I observed and interviewed baristas at local cafes, as well as home baristas, all using espresso machines to extract fresh espresso. I found that users encountered a wide range of experiences:

  • Expectations - some users expected it to be easy to use, while others were initially intimidated by the many features
  • Uses - baristas mainly used the espresso machine to make espresso-based drinks, using the steaming and hot water features to aid this process
  • Frequency of use - cafe baristas used the machine every few minutes or constantly, while home baristas use it around once per day
  • Flow of use - users moved fluidly throughout; home baristas were more careful on some steps while cafe baristas prioritized speed
  • Difficulty - basic features (eg pressing a button to extract espresso) were percieved as easy; some users found it hard to understand how additional features (eg adjusting the settings) contributed to making a higher-quality espresso shot
  • Mistakes - most mistakes made occurred during manual steps, causing the espresso to be over/under extracted; some users had issues with the water tank, which was easily fixed
